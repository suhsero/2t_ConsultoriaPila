Este es el **Proyecto Final del 2º Trimestre** para los módulos de **Lenguajes de Marcas** y **Entornos de Desarrollo**. Se trata de una plataforma web corporativa para una consultora informática ficticia, diseñada con un enfoque moderno, minimalista y de alto rendimiento.

---

## 📝 Descripción del Proyecto

**Consultoría Pila** es una landing page diseñada bajo la estética *Spotify Design*, utilizando un esquema de colores oscuros con componentes elevados. El proyecto demuestra la capacidad de integrar diseño responsive, manipulación dinámica del DOM y buenas prácticas de desarrollo.

### Secciones principales:
* **Inicio (Hero):** Presentación de alto impacto con gradientes y efectos visuales.
* **Servicios:** Renderizado dinámico de tarjetas mediante JavaScript.
* **Equipo:** Presentación de los expertos de la empresa.
* **Contacto:** Formulario interactivo con validación de estado y feedback visual.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5 Semántico:** Uso de etiquetas como `<header>`, `<main>`, `<section>`, `<article>` y `<footer>` para mejorar el SEO y la accesibilidad.
* **TailwindCSS:** Framework de utilidades CSS para un diseño 100% responsive y personalizado sin salir del HTML.
* **JavaScript (Vanilla):** * Lógica de renderizado dinámico para la sección de servicios.
    * Gestión de eventos (submit del formulario).
    * Animaciones de transiciones y estados de botones.
* **FontAwesome:** Para la iconografía profesional.
* **Google Fonts:** Tipografía sans-serif optimizada para lectura en pantallas.

## 🚀 Cómo ejecutar el proyecto

Para visualizar el proyecto en tu entorno local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/tu-usuario/consultoria-pila.git](https://github.com/tu-usuario/consultoria-pila.git)
    ```
2.  **Navegar a la carpeta:**
    ```bash
    cd consultoria-pila
    ```
3.  **Abrir el archivo principal:**
    Simplemente abre el archivo `index.html` en tu navegador preferido o utiliza la extensión *Live Server* en VS Code.
